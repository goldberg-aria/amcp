# Governance

AMCP is currently maintained as a public draft led by Nunchi AI.

## Current governance model

- protocol direction is curated by the maintainers
- draft changes are discussed through issues and pull requests
- running reference implementations are treated as important evidence

## Decision principles

- interoperability first
- minimal canonical surface
- compatibility over unnecessary breakage
- specification changes should be justified by implementation or adoption needs

## Future direction

The governance model is expected to expand as external adopters and implementers participate.

Planned future work includes:

- clearer change proposal process
- explicit versioning and deprecation policy
- broader compliance process
- multi-implementer feedback loop
